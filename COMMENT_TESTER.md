# 📲 Comment tester TimeCool sur votre téléphone — guide express

## 🚀 Méthode la plus rapide (2 minutes, zéro install)

1. **Ouvrez sur votre PC** : https://app.netlify.com/drop
2. **Glissez-déposez** le dossier `pwa/` dans la zone bleue.
3. Netlify vous donne une **URL** type `https://random-name-123.netlify.app`.
4. **Sur votre téléphone Android**, ouvrez cette URL dans **Chrome**.
5. Menu Chrome (les **⋮** en haut à droite) → **Installer l'application** (ou *Ajouter à l'écran d'accueil*).
6. ✅ L'icône **TimeCool** est sur votre écran d'accueil. Tap = app plein écran.

---

## 🛠️ Pour avoir une vraie APK (.apk) à distribuer

Suivez l'**Option 2 (GitHub Actions)** dans le `README.md`.
Résumé : créez un repo GitHub, uploadez le projet, le `.apk` se génère tout seul en 5 minutes.

---

## 📝 Ce que vous testez

C'est **exactement** la démo HTML v4.1 que vous avez validée, mais :
- En plein écran (sans la barre du navigateur)
- Avec une icône sur l'écran d'accueil
- Fonctionne hors-ligne (PWA seulement)
- Comportement de vraie app (bouton retour Android géré, pas de zoom intempestif)

Aucune fonctionnalité du brief n'est ajoutée ou retirée par rapport à la démo HTML.
